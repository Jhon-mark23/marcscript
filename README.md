# 🚀 MarcScript VPN

> All-in-One VPN Server Installer for Ubuntu/Debian VPS

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## ✨ Features

- ✅ **SSH Server** - Multi-port SSH (22, 80)
- ✅ **SSL Tunnel** - Stunnel on ports 443, 8443
- ✅ **WebSocket** - SSH over WebSocket on port 8080
- ✅ **Squid Proxy** - HTTP proxy on ports 3128, 8082, 8888
- ✅ **Xray Core** - VMess, VLess, Trojan, Shadowsocks
- ✅ **Nginx** - Web server + reverse proxy for Xray
- ✅ **User Management** - Create/trial/delete accounts
- ✅ **Easy Menu** - Simple terminal menu system

## 📋 Requirements

- Ubuntu 18.04+ or Debian 10+
- Root access
- Recommended: 1GB RAM minimum
